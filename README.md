strengthen my JavaScript fundamentals and prepare for frontend

-   JavaScript Basics
-   Variables

-   Operators

-   Functions

-   Arrays

-   Practice Programs

## Important Concepts

The repository focuses on understanding and practicing core JavaScript
concepts such as:

-   `var`, `let`, and `const`
-   Primitive and Non-Primitive Data Types
-   Conditional Statements
-   Loops
-   Regular and Arrow Functions
-   Function Scope and Block Scope
-   JavaScript Objects
-   Arrays and Array Operations
-   `forEach()`
-   `map()`
-   `filter()`
-   `reduce()`
-   `find()` and `findIndex()`
-   `slice()` and `splice()`
